# _murex_ Shell Docs

## Optional Command Reference: `gz` 

> Compress or decompress a gzip file

## Description

An optional builtin for compressing or decompressing a gzip stream from STDIN.

## Usage

    <stdin> -> gz -> <stdout>
    
    <stdin> -> !gz -> <stdout>

## Synonyms

* `gz`
* `!gz`
